# 쇼핑 리스트 앱

간단한 쇼핑 리스트 웹 애플리케이션입니다. 로컬 브라우저에서 실행되며, 로컬 스토리지를 사용하여 데이터를 저장합니다.

## 기능

- ✅ 항목 추가: 입력창에 물품을 입력하고 추가 버튼 클릭
- ✅ 항목 삭제: 각 항목 옆의 휴지통 아이콘 클릭
- ✅ 구매 완료 체크: 체크박스를 클릭하여 구매 완료 상태 변경
- ✅ 필터링: 전체 보기 / 구매 예정 / 구매 완료 항목 필터링
- ✅ 완료된 항목 일괄 삭제
- ✅ 통계 표시: 전체 항목, 구매 완료, 남은 항목 수
- ✅ 로컬 스토리지: 브라우저를 닫아도 데이터 유지
- ✅ 반응형 디자인: 모바일, 태블릿, 데스크톱 지원
- ✅ 시각적 피드백: 알림 메시지 표시

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다.
2. 입력창에 구매할 물품을 입력하고 "추가" 버튼을 클릭하거나 Enter 키를 누릅니다.
3. 항목 왼쪽의 체크박스를 클릭하여 구매 완료 상태를 변경합니다.
4. 항목 오른쪽의 휴지통 아이콘을 클릭하여 항목을 삭제합니다.
5. 상단의 필터 버튼을 사용하여 원하는 항목만 볼 수 있습니다.
6. "완료된 항목 삭제" 버튼으로 구매 완료된 항목을 한 번에 삭제할 수 있습니다.

## 실행 방법

### 웹 애플리케이션 실행
1. 이 폴더의 `index.html` 파일을 더블클릭하거나
2. 웹 브라우저에서 파일을 드래그 앤 드롭하거나
3. 터미널에서 다음 명령어 실행:
   ```bash
   # Windows
   start index.html
   
   # Mac
   open index.html
   
   # Linux
   xdg-open index.html
   ```

### 개발 환경 설정
```bash
# 의존성 설치
npm install

# 테스트 실행
npm test

# 헤드리스 모드 테스트 실행
npm run test-headless
```
